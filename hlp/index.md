# ApgMng Help


<br>

### Import the library in `monorepo.ts` of your project

Using:

```Typescript
export * as Mng from "https://raw.githubusercontent.com/Pangeli70/Mng/master/mod.ts";
```

Import in your file using:

```Typescript
import { Mng, ... } from "[...PATH...]/deps.ts";
```
